Configuration Notes
===================

I.  Setup on 32bit Systems
--------------------------
1. Download Portable Python 2.7 (32bit) from http://portablepython.com/wiki/Download/ and extract contents to <project-root>\python_x86
2. Run <project-root>\setup\setup-win32.bat to configure required Python packages
3. Run <project-root>\run_x86.bat to test installation :-)

II. Setup on 46bit Systems
--------------------------
1. Download Portable Python 2.7 (64bit) from http://portablepython.com/wiki/Download/ and extract contents to <project-root>\python_x64
2. Run <project-root>\setup\setup-win64.bat to configure required Python packages
3. Run <project-root>\run_x64.bat to test installation :-)

Additionally, wxpython should be installed, from http://www.wxpython.org/download.php
Additionally, pywin32 should be installed, from http://starship.python.net/crew/mhammond/win32/Downloads.html
